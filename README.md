# Development Practices

This repository contains exercises and information about Test-Driven Development (TDD) and Acceptance Test-Driven Development (ATDD).

## Documentation

- ATDD.md: Brief explanation of Acceptance Test-Driven Development and Behavior-Driven Development (BDD).
- TDD.md: Detailed explanation of Test-Driven Development, its benefits, and practices.

## Exercises

### Exercise 1: Palindrome Checker
Located in the ex1/ directory. It contains a function to check if a string is a palindrome.

To run the tests for Exercise 1:
```bash
python3 -m unittest discover ex1
```

### Exercise 2: Bank Account System
Located in the ex2/ directory. It contains a BankAccount class with features like deposit, withdrawal, and transfers, including fraud detection.

To run the tests for Exercise 2:
```bash
python3 -m unittest discover ex2
```
